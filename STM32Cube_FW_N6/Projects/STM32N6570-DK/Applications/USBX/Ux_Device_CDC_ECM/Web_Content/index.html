The The STM32N6570-DK Discovery kit is a complete demonstration and development platform for the STMicroelectronics Arm® Cortex®-M55 core-based STM32N657X0H3Q microcontroller. Arm® 32-bit Cortex®-M55, 3360 CoreMark®, frequency up to 800 MHz, 1280 DMIPS, 32-Kbyte ICACHE, 32-Kbyte DCACHE. Arm® MVE (M-Profile vector extension) and Arm® TrustZone® MPU, NVIC. Single and half-precision floating point unit.


The STM32N6570-DK Discovery kit is used as a reference design for user application development before porting to the final product, thus simplifying the application development. The full range of hardware features available on the board helps users to enhance their application development by an evaluation of all the peripherals (USB Type-C®, Octo-SPI Flash memory device,

 
Hexa-SPI SRAM memory device, audio codec, digital microphones, Ethernet, ADC, flexible extension connectors, user buttons). ARDUINO Uno V3, PmodT and STMod+ connectors provide easy connection to extension shields or daughterboards for specific applications. The STM32N6570-DK Discovery kit integrates an STLINK-V3EC embedded in-circuit debugger and programmer for the STM32 MCU, with an USB Virtual COM port bridge and comes with the comprehensive STM32CubeN6 MCU Package, which provides an STM32 comprehensive software HAL library as well as various software examples.

 About this application

 

This application is part of the Azure RTOS STM32Cube expansion package for STM32N6 series, it presents HTTP webserver developed based on the USBX CDC ECM class, Azure RTOS NetXDuo TCP/IP network stack and FileX file system stack, it allows the loading of a static home web page from sd card and led control web page.

Application main features:

  • Azure RTOS USBX
      • Class: Device CDC ECM.
  • Azure RTOS NetXDuo
      • Protocols: IP, ICMP, ARP, UDP, TCP.
      DHCP client (dynamic IP address allocation).
      Addons/Web HTTP.
  • Azure RTOS FileX API (to fetch the html page from sd card).


For more information you can refer to Azure RTOS documentation  Azure RTOS USBX & Azure RTOS NetXDuo & Azure RTOS FileX
