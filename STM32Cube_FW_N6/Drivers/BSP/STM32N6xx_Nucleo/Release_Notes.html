Release Notes for

STM32N6xx_Nucleo

Copyright Â© 2023 STMicroelectronics

Purpose

This driver provides a set of functions to manage:

  • LEDs, push-button, Virtual Com Port (VCP), USB-PD and external memory access on STM32N6xx_Nucleo Kit from STMicroelectronics

Update History

Main Changes

Patch release of STM32N6xx_Nucleo BSP board drivers

Contents

  • Changes compared with V0.3.0 release
    • Add Nucleo revC board support
      • Add external SMPS initialization function BSP_SMPS_Init()
    • Correct BSP_XSPI_NOR_DeInit() function to ensure XSPI peripheral is unclocked after registers reset

Dependencies

None

Notes

stm32n6xx_nucleo_conf_template.h file must be copied in user application as stm32n6xx_nucleo_conf.h with optional configuration update
