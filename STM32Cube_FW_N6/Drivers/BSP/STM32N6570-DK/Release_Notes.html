Release Notes for

STM32N6570-DK

Copyright © 2023 STMicroelectronics

Purpose

This driver provides a set of functions to manage:

  • LEDs, push-button, Virtual Com Port (VCP), Audio, USB-PD and external memories access on STM32N6570-DK Kit from STMicroelectronics

Update History

Main Changes

Patch release of STM32N6570-DK BSP board drivers

Contents

  • Changes compared with V0.5.0 release
    • Add STM32N6570-DK revC board support
      • Add external SMPS initialization function BSP_SMPS_Init()
      • Change camera I2C instance from I2C2 to I2C1
    • Correct BSP_XSPI_NOR_DeInit() function to ensure XSPI peripheral is unclocked after registers reset

Known Limitations

None

Dependencies

None

Notes

Enable STM32N6570_DK_REV=STM32N6570_DK_A01 compilation switch for BSP driver running on STM32N6570-DK MB1939 revA board.
Enable STM32N6570_DK_REV=STM32N6570_DK_B01 compilation switch for BSP driver running on STM32N6570-DK MB1939 revB board.

Main Changes

Third beta release of STM32N6570-DK BSP board drivers

Contents

  • Changes compared with V0.4.1 release
    • Update stm32n6570_discovery_usbpd_pwr.c/.h to support Discovery board rev B01 (updates of pins connected to tcpp03 component)
    • Remove RIF and SRAM configuration from LTDC_MspInit()
    • Update LCD_LAYER_0_ADDRESS and LCD_LAYER_1_ADDRESS in stm32n6570_discovery_conf_template.h to point at SRAM1 area
    • Add isp_param_conf.h and update stm32n6570_discovery_camera.c/h to support IMX335 camera sensor
    • Add stm32n6570_discovery_camera_ov5640.c/h to maintain OV56540 camera sensor support

Known Limitations

None

Dependencies

None

Notes

Enable USE_STM32N6570_DK_REV_B01 compilation switch for BSP driver running on STM32N6570-DK MB1939 rev B01 board.

Main Changes

Patch release of STM32N6570-DK BSP board drivers

Contents

  • Changes compared with V0.4.0 release
    • Add BSP_CAMERA_Suspend() and BSP_CAMERA_Resume() APIs
    • Align stm32n6570_discovery_camera.c/.h with latest HAL DCMIPP version
    • Limit RIF configuration in stm32n6570_discovery_camera.c to secure mode

Known Limitations

  • Image quality when resorting to IMX335 camera isn’t optimized yet and will be improved in next releases. User can resort to OV5640 camera sensor in the meantime.

Main Changes

Second beta release of STM32N6570-DK BSP board drivers

Contents

  • Changes compared with V0.3.2 release
    • Remove characterization-related code from USB-PD BSP driver
    • Update camera BSP driver
      • Remove pipe selection and only use PIPE1
      • Add register callback feature
      • Add downsize configuration for IMX335 camera sensor
      • Add first version of dynamic image correction to camera BSP (will be improved in next releases)
      • Align code with latest HAL DCMIPP driver that integrates CSI functions formerly provided in HAL CSI driver
      • Update clock configuration
    • Align LCD BSP driver with HAL LTDC latest version
