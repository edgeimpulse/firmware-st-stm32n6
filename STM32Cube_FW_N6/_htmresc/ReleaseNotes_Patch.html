STM32CubeN6 Firmware Package V0.5.1 / 26-July-2024

Main Changes

Official Patch Release of STM32CubeN6 Firmware package supporting STM32CubeN6 devices

This patch targets the following points related to CMSIS, Nucleo and DK BSP drivers, examples and applications:

- CMSIS update to include fix and optimizations

- HAL DCMIPP update to include fix

- DK BSP update to support STM32N6570-DK revC board

- Nucleo BSP update to support NUCLEO-N657X0-Q revC board

Examples and applications

- Update stm32n6570_discovery_conf.h files to support revC board
